QUICK DEPLOY — 2 OPTIONS
================================================================================

OPTION 1: VERCEL (EASIEST - 5 MINUTES)
────────────────────────────────────────
1. Create GitHub repo (if not already)
   git remote add origin https://github.com/YOUR-USERNAME/censuscat.git
   git push -u origin main

2. Go to vercel.com
3. Click "Import Project"
4. Select your repo
5. Click "Deploy"
6. Get live URL! 🎉

Auto-deploys on every git push.


OPTION 2: YOUR SERVER (SELF-HOSTED)
────────────────────────────────────
1. On your server:
   cd /opt
   git clone YOUR-REPO censuscat
   cd censuscat
   npm install
   npm run build

2. Run with PM2:
   npm install -g pm2
   pm2 start "npm start" --name censuscat
   pm2 startup
   pm2 save

3. Set up Nginx reverse proxy to port 3000

4. Add SSL certificate (Let's Encrypt)

See: DEPLOYMENT-SERVER.md for details
