# The Census Cat Mystery

A fully functional interactive website documenting the mysterious cat sightings at the US Census Bureau headquarters.

## Features

✅ **Hero Section** — Dramatic introduction to the mystery  
✅ **Unanswered Questions** — Expandable mystery cards  
✅ **Building Layout** — Interactive floor maps & sighting hotspots  
✅ **Sighting Report Form** — Detailed citizen reporting system  
✅ **Cat Naming Vote** — Official voting + custom suggestions  
✅ **Policy Implications** — Humorous policy review scenarios  
✅ **Official Footer** — Workhorse Collective credits & contact  

## Project Context

This website demonstrates the viability of using **Claude Code and agentic programming** to rapidly create specialized applications for government agencies.

## Tech Stack

- **Next.js 14** (Pages Router)
- **React 19** with TypeScript
- **Tailwind CSS** (custom styling)
- **Client-side state** (sightings & votes)

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

Deploy to Vercel:

```bash
vercel deploy
```

Or connect GitHub repo for auto-deploy on push.

## Customization

All components are in `src/components/`:
- `Header.tsx` — Top navigation
- `Hero.tsx` — Hero section
- `Mystery.tsx` — Expandable questions
- `BuildingMap.tsx` — Sighting locations
- `SightingForm.tsx` — Report submission
- `CatNames.tsx` — Voting system
- `PolicyImplications.tsx` — Policy updates
- `Footer.tsx` — Credits & contact

Edit text, colors, or functionality as needed.
